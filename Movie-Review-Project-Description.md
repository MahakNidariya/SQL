# 🎬 Movie Review SQL Project

This beginner SQL project uses a simple movie review dataset to demonstrate core SQL commands. It simulates a basic movie review platform where users rate and comment on movies.

## 📁 Tables

- `movies` – Movie details
- `users` – Reviewer profiles
- `reviews` – Ratings and comments per movie per user

## 💡 SQL Concepts Used

- `SELECT`, `FROM`, `WHERE`, `LIKE`
- `GROUP BY`, `HAVING`
- `ORDER BY`, `LIMIT`
- Aggregates: `AVG`, `COUNT`, `MAX`, `MIN`
- `JOIN` between multiple tables

## 🔍 Sample Insights

- Movies with the highest average ratings
- Most active countries for reviews
- Latest reviews by users
- Comments containing keywords like "amazing"
- Average rating per movie or genre

## 📎 How to Use

1. Run `schema.sql` to create the tables
2. Run `data.sql` to populate the tables
3. Run `queries.sql` to explore and analyze
